## Research & Publications
- **The Halt Vector: Internalizing a Causal Steering Intervention for Efficient Reasoning** — *first author, COLM 2026.* Found a causal "halt vector" in a 7B reasoning model and internalized it into the weights, cutting ~24% of thinking at held accuracy with no inference-time hooks. [code + paper](https://github.com/dylanjayabahu/halt-vector)
- **Deep Learning Model for Invasive Ductal Carcinoma Detection in Histopathology Images** — *sole author, IEEE CCECE 2025.* [repo](https://github.com/dylanjayabahu/idc-detection) · [IEEE Xplore](https://ieeexplore.ieee.org/document/11364431)
- **Human Action Detection using FMCW mmWave Radar** — *first author, CVIS 2024.* [JCVIS](https://openjournals.uwaterloo.ca/index.php/vsl/article/view/6364)

## Projects
